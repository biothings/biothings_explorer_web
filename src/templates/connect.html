How it works

There might not exist direct evidence implying a relationship between A and B. However, if both A and B is known to be related to concept X, it might infer a certain relationship between A and B. This tool aims at finding the intermediate concept X which servers as a bridge connecting A and B.

An Example

We don't know how DRUG (B) can be used to treat DISEASE (A). However, if we know that DRUG (B) targets GENE (X) and mutations in GENE(X) is known to cause DISEASE(A). Then, we might infer that DRUG(B) might be used to treat DISEASE(A)
